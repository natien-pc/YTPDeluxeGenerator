This folder is intended to hold your legacy-style assets for YTP Deluxe Generator.

Place your own media here; do NOT commit copyrighted OS files to public repos.

Suggested filenames (placeholders):
- error_winxp.png        <- old error dialog screenshot or stylized overlay
- error_vista.png
- advert_2009.png        <- old banner/ad overlay
- advert_2012.png
- os_sound_buzz.wav      <- old system beep / shutdown sound
- os_sound_error.wav
- meme_overlay.png
- explosion_1.png

The engine will scan this directory and pick assets automatically if you leave the GUI asset fields empty.
